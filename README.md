# 🏫 Projeto Escola

> Uma aplicação web simples e interativa desenvolvida com tecnologias web fundamentais.

Este projeto foi construído para simular ou apresentar páginas relacionadas ao ambiente escolar, integrando uma interface visual estilizada com comportamentos dinâmicos no navegador.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação dos elementos da página (`escola.html`).
- **CSS3:** Estilização, layout e responsividade do sistema (`escola.css`).
- **JavaScript:** Funcionalidades lógicas e interatividade do usuário (`escola.js`).

## 📦 Estrutura do Repositório

O projeto é composto pelos seguintes arquivos principais:
* `escola.html` - Arquivo principal contendo a estrutura da página.
* `escola.css` - Folha de estilos responsável pelo design visual.
* `escola.js` - Scripts de interatividade da página.
* `adolescente deitada.webp` - Imagem otimizada utilizada na interface do usuário.

## 🚀 Como Executar o Projeto

Você não precisa instalar nenhum framework ou gerenciador de pacotes para rodar este projeto. Siga os passos:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com
   ```
2. **Acesse a pasta:**
   ```bash
   cd Projeto-Escola/"Projeto Escola"
   ```
3. **Abra o arquivo no navegador:**
   Dê um duplo clique no arquivo `escola.html` para visualizar a página funcionando instantaneamente.
